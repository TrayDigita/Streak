# STREAK!


## Installations

```bash
composer create-project traydigita/streak --stability dev example.com
```


Copy [Example.Config.php](Example.Config.php) to project root.


## Requirements

- Php `8.1` or `8.2`
- Json Extension
- cURL
- Pdo MySQL
- Filter Extension
- OpenSSL
